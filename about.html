About Claude Code Tutorial

Our Mission

We believe every developer should have access to high-quality, practical AI coding education. Claude Code Tutorial provides free, bilingual, in-depth lessons that take you from complete beginner to expert CLI agent user. All content is original, hand-crafted, and continuously updated.

Who This Is For

Whether you're a senior engineer exploring AI-assisted development, a junior developer looking to accelerate your learning, or a team lead evaluating tools for your organization — these lessons are built for you. We cover fundamentals for newcomers and advanced patterns for experienced users.

What You'll Learn

  • Install and configure Claude Code on any platform
  • Master all 15+ slash commands and their use cases
  • Write effective CLAUDE.md files that make Claude smarter
  • Configure permissions for the right balance of safety and productivity
  • Build automation hooks that enforce team standards
  • Create custom skills for repeatable, complex workflows

Site Features

32 in-depth lessons across 6 categories. Bilingual support — English-first with full Chinese translations. Lightweight, fast, and mobile-responsive. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools.
